# 基金定投策略回测工具

一个本地桌面 GUI（Python + Tkinter），让你自由配置 “下跌/上涨多少就买/卖多少” 的规则，然后用 akshare 抓取真实基金净值，回测 1月 / 3月 / 6月 / 1年 / 3年 区间内的总收益。

## 安装

```bash
pip install -r requirements.txt
```

Tkinter 是 Python 自带的，无需额外安装。Windows 下中文图表字体使用 `Microsoft YaHei`。

## 运行

```bash
python app.py
```

## 使用

1. **添加基金**：在 “基金代码” 输入框输入 6 位代码（如 `012349`），点击 “获取最新数据”。完成后该基金会出现在 “已保存基金” 下拉框中，下次直接选择即可，无需重新输入。本地缓存位于 `data/<code>.csv`。
2. **编辑策略**：默认已经预置了你描述的 “跌2%买50 / 跌3%买100 / 跌4%买200” 三条规则。点 “新增 / 编辑 / 删除” 按钮可自由修改。每条规则字段：
  - `名称`：自定义。
  - `频率`：`daily`（每个交易日）/ `weekly`（每周指定星期）/ `monthly`（每月指定日期，非交易日顺延到下一个交易日）。
  - `星期`：0=周一 … 6=周日（仅 weekly 生效）。
  - `月日`：1–28（仅 monthly 生效）。
  - `方向`：`drop`（跌）/ `rise`（涨）。
  - `阈值%`：日涨/跌幅达到该百分比时触发。
  - `操作`：`buy`（买入）/ `sell`（卖出）。
  - `金额`：本次买入或卖出的金额（元）。
3. **运行回测**：选择回测区间，点击 “运行回测”。
  - 左侧显示策略汇总（累计买入、持仓市值、收益率、最近交易等）。
  - 右侧绘制 “策略总价值” 与 “净投入累计” 曲线。

## 规则匹配语义

- 同一交易日，若多条 **同方向（都是买入或都是卖出）** 规则同时满足，**只触发阈值最大的那一条**。这样写 “跌2%买50 / 跌3%买100 / 跌4%买200” 时，跌 5% 当日只会触发 200 元那一条，避免重复下单。
- 不同方向（买入 + 卖出）的规则相互独立，可同日触发。
- 卖出金额会被自动限制为当前持仓市值，避免超卖。

## 数据来源

通过 [akshare](https://akshare.akfamily.xyz/) 的 `fund_open_fund_info_em` 接口获取场外开放式基金的单位净值。

## 文件结构

```
app.py              # 主程序
funds.json          # 已获取基金列表 {code: name}
strategies.json     # 当前策略规则
data/<code>.csv     # 各基金的本地净值缓存
data/_all_funds.csv # 基金全名表缓存（用于回填基金名）
```
